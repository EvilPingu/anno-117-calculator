# TypeScript Migration for Anno 1800 Calculator

## Quick Start

This project has been set up for TypeScript migration. Here's how to get started:

### 1. Install Dependencies

```bash
npm install
```

### 2. Run the Migration Script

```bash
npm run migrate
```

This will create TypeScript versions of all JavaScript files in the `src/` directory.

### 3. Check TypeScript Compilation

```bash
npm run type-check
```

### 4. Build the Project

```bash
npm run build
```

## Common Issues

### 1. Type Errors
- Use `(ko as any)` for Knockout extensions
- Add proper type declarations for global objects
- Use type assertions where necessary

### 2. Import/Export Issues
- Keep `require()` for Knockout and AMD modules
- Use ES6 imports for TypeScript modules
- Handle webpack context with `(require as any).context()`

### 3. Global Object Access
- Declare global interfaces in `types.ts`
- Use `(window as any)` for global assignments
- Add proper type guards
